# Cookie Clicker

Ein kleines Cookie-Clicker-Spiel als einzelne HTML-Datei.

## Starten

Oeffne `index.html` direkt im Browser.

## Dateien

- `index.html` - Spieloberflaeche, Styles und Spiellogik

## Nutzung

Klicke auf den Cookie, sammle Punkte und kaufe Upgrades, um schneller Cookies zu erzeugen.
